**Python Backend Developer** · LLM systems · AI infrastructure

## About Me

I build production-ready AI systems on the backend:

- Deploy and serve local LLMs (Ollama) in containerized environments, expose them via clean REST APIs
- Build RAG pipelines with vector search (ChromaDB, pgvector) for document retrieval and knowledge bases
- Design AI agents — single-step, multi-step, and multi-agent systems that reason and act
- Integrate third-party AI APIs (OpenAI, Anthropic) into backend services

---

## Selected Systems

**LLM Gateway** — FastAPI service with multi-provider routing (OpenAI, Anthropic, Ollama).
Single interface for switching between models without changing client code.

**RAG Pipeline** — document retrieval with pgvector / ChromaDB, embedding layer, context-aware responses via LLM.

**Agent System** — multi-step agents with tool use and planning. Supports single and multi-agent scenarios.
